## 干货（GankApp）

干货集中营Android客户端

由衷感谢代码家在[「干货集中营」](http://gank.io/)提供的干货和福利，感谢所有干货的作者的辛勤付出。

该项目使用MVP+RxJava2+Retrofit2实现。
